# Ros
## Preamble
### Tech Stack
* Ros 2 (Jazzy)
* Gazebo (Jetty/ionic/Harmonic)
* PX4
* TensorFlow

### Background
Ros 2 using rolling releases. 
We are using **Jazzy**, which is for Ubuntu 24 (noble). 
Gazebo Harmonic is recommened for Jazzy. Ionic is the inbetween version. Jetty is the new version for it, but it's under development still.

[Release list](https://docs.ros.org/en/rolling/Releases.html)
[Gazebo](https://gazebosim.org/docs/all/getstarted/)
